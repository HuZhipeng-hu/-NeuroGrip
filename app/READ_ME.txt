🌐 项目简介
本项目面向 NeuroGrip Pro 智能仿生机械臂设计，聚焦“以用户为中心的智能交互体验”，通过直观的数据可视化、模拟肌电信号反馈和个性化训练引导，为截肢用户提供高效、自然的义肢操控与康复训练管理界面。

作为本次比赛的前端部分，本 App 已完成全部页面开发与核心功能逻辑实现，支持设备连接状态监控、手势训练控制、数据统计分析、定位追踪及系统设置等功能，具备良好的可扩展性与用户体验。

🔗 后端服务将在复赛阶段接入 Spring Boot + 华为云 IoTDA / ModelArts / GaussDB 实现全链路打通。

开发环境：DevEco Studio 5.0.5 + HarmonyOS SDK API 10（Stage 模型）

- UI 实现：完整实现首页、训练、数据、我的、设置、定位等 6 大模块的页面布局与交互逻辑  
  • 所有图表（趋势图、进度环、波形图）均基于 ArkUI 基础组件与 Canvas 自主实现

功能模块
1. 首页 (Index)
首页展示了仿生机械臂的状态卡片、设备区、训练建议等信息，使用户能够快速了解当前设备状态和训练进展。

2. 训练 (Training)
训练页面提供了一个全面的训练环境，包括手势库选择、实时波形图显示、灵敏度调节以及训练进度跟踪等功能。用户可以在此进行仿生机械臂的各种训练，并根据反馈调整训练参数。

3. 数据 (DataPage)
数据页面展示了一周内每日抓握次数的统计数据及图表分析，有助于用户理解自己的训练趋势和效果。

4. 我的 (Profile)
“我的”页面集成个人资料展示、数据卡片与功能入口，用户可查看使用时长、设备信息、定位记录等，并快速跳转至相关功能模块。

5. 设置 (Setting)
设置页面允许用户对应用的功能进行个性化配置，如手势自定义、数据同步、联系客服等。此外，还提供退出登录功能。

6. 定位 (Location)
定位页面显示了设备的最后连接位置的地图视图，便于用户追踪设备的位置。

组件说明
CustomAppBar: 自定义顶部栏组件，支持标题显示、返回按钮及设置按钮。
BottomNavBar: 底部导航栏组件，用于在首页、训练、数据和个人中心之间切换。
DeviceCard: 设备卡片组件，展示设备连接状态、信号强度、电池电量等信息。
TrainingProgress: 训练进度条组件，展示当前训练进度及历史最佳成绩。
StableBarChart: 稳定的条形图组件，用于展示每周每日抓握次数的数据。
SettingItem: 设置项组件，用于构建功能设置列表中的各个选项。




使用指南
应用启动后默认进入首页，展示仿生机械臂的基本信息和训练建议，查看仿生机械臂的基本信息和训练建议。
若要开始或继续训练，请点击底部导航栏上的“训练”图标，进入训练页面选择手势并调整相关参数。
在数据页面，您可以查看详细的训练数据和趋势分析，以评估您的训练效果。
转到“我的”页面，您可以访问更多功能选项，例如设备管理、使用时长统计、设备定位等。
通过“设置”页面，您可以根据需要自定义应用的功能，确保最佳用户体验.
